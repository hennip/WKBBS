# SRM

This folder contains scripts and data related to the development of the Southern River model to include rivers from AU5 and AU6.

## Instructions
1. Open 'setup_SRM' and select rivers to include in model
2. Open 'run_SRM' and run line by line for rivers included in setup_SRM and to use custom selections 

## Contents
- `model_SRM.R`: original model
- `model_SRM_parr`: SRM model including parr 0+ size
- `data_SRM` : script to prepare data
- `setup_SRM`: script to run model
- `run_SRM`: script to run model
- `data´: data to run the model (this data will be moved to more appropriate place later)
